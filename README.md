# Xerox-946.github.io

基于 **Angular 20** 与 **Angular Material** 重构的防疫宣传专题站。

## 开发

```bash
npm install
npm start
```

## 构建

```bash
npm run build
```

## GitHub Pages 构建

仓库当前发布目标为 `https://hangjinlei.github.io/Xerox-946.github.io/`，因此使用项目页基址：

```bash
npm run build:pages
```

站点使用了 **hash 路由**，因此在 GitHub Pages 上刷新子页面也能正常访问。
